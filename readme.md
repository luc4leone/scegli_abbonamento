# Scegli Abbonamento Pound4Pound

Questo progetto contiene le pagine HTML e il codice JavaScript per calcolare e mostrare i prezzi degli abbonamenti Pound4Pound in base alle configurazioni selezionate.

## Struttura del progetto

- `abbonamento_v2.html` - Pagina per l'abbonamento V2
- `abbonamento_v3.html` - Pagina per l'abbonamento V3
- `app.js` - Logica di calcolo e rendering
- `styles.css` - Stili CSS
